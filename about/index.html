About

What this is and why it exists.

I kept needing simple online dev tools — code editors, formatters, testers — and everything out there was bloated with ads, account walls, and pop-ups. So I started building clean versions for myself and figured I'd share them.

Every tool runs entirely in your browser. Nothing is stored on a server. Nothing is tracked. The site is hosted on GitHub Pages at zero cost.

Analytics

The site uses Plausible Analytics for basic traffic stats. It's cookie-free and doesn't collect personal data. That's it.
