Privacy Policy

Last updated: March 2026

takeonanything.com is built to respect your privacy. Here's the short version: we don't collect your personal data, we don't set cookies, and we don't track you across the web.

What we don't collect

  • No personal information (name, email, IP address)
  • No cookies of any kind
  • No browser fingerprinting
  • No third-party tracking scripts
  • No advertising pixels or retargeting

Your code stays in your browser

All tools on this site run entirely in your browser. Code you write or paste into any tool is never sent to a server. If a tool uses localStorage (for example, to remember your preferences or save your work between sessions), that data lives only on your device and is never transmitted anywhere.

Analytics

We use Plausible Analytics to understand basic site traffic — things like page views and referral sources. Plausible is a privacy-friendly, cookie-free, GDPR-compliant analytics tool. It does not collect personal data, does not use cookies, and does not track you across sites. All data is aggregated and anonymous. You can read Plausible's data policy for full details.

Third-party resources

Some pages load fonts from Google Fonts and code editor libraries from Cloudflare's CDN. These services may log standard server access information (such as IP addresses) according to their own privacy policies. We have no control over or access to that data.

Shareable links

Some tools offer a "Share" feature that encodes your code directly in the URL. This data is not stored on any server — it lives entirely in the link itself. Anyone with the link can see the shared code, so don't share links containing sensitive information.

Changes to this policy

If this policy changes, the updated version will be posted here with a new "last updated" date. No prior notice is required, but we'll keep it honest.

Contact

Questions about this policy? Reach out at hello@takeonanything.com.
