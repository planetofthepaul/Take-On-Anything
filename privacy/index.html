Privacy Policy

Last updated: March 2026

Your code stays in your browser

All tools on this site run entirely in your browser. Code you write or paste is never sent to a server. If a tool uses localStorage (to save preferences or your work between sessions), that data lives only on your device.

No cookies, no tracking

This site does not set cookies, does not use browser fingerprinting, and does not run any third-party tracking or advertising scripts.

Analytics

We use Plausible Analytics for basic traffic stats (page views, referral sources). Plausible is cookie-free, GDPR-compliant, and does not collect personal data. You can read their data policy for details.

Third-party resources

Some pages load fonts from Google Fonts and libraries from Cloudflare's CDN. These services may log access information according to their own policies.

Shareable links

Some tools let you share code via URL. The code is encoded directly in the link — nothing is stored on a server. Anyone with the link can see the code, so don't share links containing sensitive information.

Changes

If this policy changes, the updated version will be posted here with a new date.
