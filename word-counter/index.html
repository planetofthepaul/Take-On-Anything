What This Counts

This tool counts words, characters, characters without spaces, sentences, and paragraphs. It also gives a simple reading-time estimate based on word count. That makes it useful for essays, blog posts, captions, product descriptions, emails, resumes, and anything else with a length limit.

Word Count and Character Count in One Place

Some tools only show a word count. Some only show character count. This one does both and updates instantly while you type or paste. If you need to stay under a character limit, trim an article to a target length, or check how long something is before posting it, this gives you the numbers quickly.

How It Works

Word count is based on text separated by spaces. Character count includes everything in the box, including spaces and line breaks. Characters without spaces removes all whitespace first. Sentence count uses ending punctuation like periods, question marks, and exclamation points. Paragraph count is based on blank lines between blocks of text. Reading time is estimated from total word count using 200 words per minute.
